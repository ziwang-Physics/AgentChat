🐍 Python SDK

agentchat 包 — 异步会话管理 · 结构化输出 · 多模态输入

快速开始

from agentchat import GeminiSession

async with GeminiSession() as gs:
    result = await gs.ask("帮我分析近期的科技股走势")
    print(result.provider_used)  # 'Gemini'
    print(result.model_used)     # 'Pro Extended'
    print(result.time_seconds)   # 156.0

API 一览

方法功能返回
gs.ask(prompt)发送 prompt，走完整降级链AskResult (含 provider/model/time/chain)
gs.chat(prompt)多轮对话，复用已有 tab同上
gs.ask_structured(prompt, schema)Pydantic 结构化输出 + 3 次重试Pydantic 实例
gs.ask_with_image(prompt, path)图片通过系统剪贴板粘贴到 GeminiAskResult
gs.batch(prompts)并行批量推理，N 个 tab 并发list[AskResult]

结构化输出

from pydantic import BaseModel, Field

class WeatherReport(BaseModel):
    city: str
    temp: float = Field(ge=-50, le=55)
    humidity: int = Field(ge=0, le=100)

async with GeminiSession() as gs:
    report = await gs.ask_structured("查询北京天气", schema=WeatherReport)
    print(report.temp)  # 25.0 (float, 已验证)

📦 AskResult

@dataclass
class AskResult:
    response: str          # AI 回复
    provider_used: str     # 'Gemini'
    model_used: str        # 'Pro Extended'
    fallback_chain: list   # 降级路径
    total_time_ms: int     # 耗时
    success: bool          # 是否成功
