Claude Desktop 配置

{
  "mcpServers": {
    "agentchat": {
      "command": "node",
      "args": ["path/to/skills/mcp-server/index.mjs"]
    }
  }
}

注意：MCP Server 使用 StdioServerTransport（stdio 模式），Claude Desktop 通过 stdin/stdout 与它通信。不需要额外启动 HTTP 服务器。
